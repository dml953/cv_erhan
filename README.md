# cv_erhan

## Outil ajoute: Assistant Vinted (vente rapide)

Un outil HTML autonome a ete ajoute pour aider a preparer des annonces Vinted rapidement:

- genere un titre d'annonce
- propose une description prete a coller
- calcule un prix de publication + prix plancher de negociation
- fournit des messages types pour repondre vite aux acheteurs
- donne une checklist photo + un mini planning de republication

### Utilisation

Ouvre simplement ce fichier dans un navigateur:

`vinted-assistant.html`

> Note: cet assistant n'automatise pas la publication sur Vinted. Il sert a gagner du temps tout en restant dans un usage propre.

## Outil ajoute: Bot bonnes affaires vehicules (multi-plateformes)

Un second outil HTML autonome a ete ajoute pour analyser des annonces auto depuis:

- Leboncoin
- Facebook Marketplace
- AutoScout
- mobile.de

Fonctions principales:

- estimation d'une cote de marche
- calcul des frais potentiels (distribution, embrayage, pneus, freins, etc.)
- cout total reel (prix + frais + import mobile.de)
- score "bonne affaire" et verdict (excellent / bonne affaire / a negocier / surcote)
- export CSV des annonces triees

### Utilisation

Ouvre ce fichier dans un navigateur:

`auto-deal-finder.html`

> Note: cet outil n'automatise pas le scraping de plateformes. Il sert de radar d'analyse pour comparer rapidement des annonces et prioriser les meilleures.
